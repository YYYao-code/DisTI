# DisTI
This repository supplements our paper “DisTI: Multivariate Time Series Anomaly Detection via Disentangled Temporal and Inter-metric Pattern Mining” (under review for 《Advanced Engineering Informatics》)

run ```python main.py``` to train models on relevant datasets
